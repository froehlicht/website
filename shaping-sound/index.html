Shaping Sound: A Comparative Study of Input Devices for Music Production and Performance

Bachelor Thesis - Digital Media (Multimedia Engineering)

This thesis investigates different input methods for music production through the development of a Motion Control plugin and a comparative user study between Motion Control, MIDI Controller, and GUI-based parameter control.

Research Components

Research Approach

Comparative analysis of three input methods: Motion Control, Hardware Controller, and GUI-based control

Practical Implementation

Development of a VST3/AU plugin with YOLO-based hand tracking for empirical evaluation

User Study

Empirical evaluation of precision and usability across different input methods

Key Plugin Features

Advanced Control Features

Simultaneous 4-parameter control (both hands, X/Y axes) • Individual mute/solo system for each control channel • Configurable MIDI CC mapping (0-127) • Spatial interaction zones for precise control

User Experience Innovations

No additional hardware required - works with standard webcam • Real-time visual feedback with hand tracking overlay

Professional DAW Integration

Native VST3/AU plugin format (not external software) • Works with all major DAWs (FL Studio, Ableton Live)

Real-time Performance Features

Low-latency MIDI generation optimized for live performance • Adaptive smoothing algorithm for stable parameter control • Multi-threaded architecture for consistent real-time processing

Technologies & Methods

JUCE Framework ONNX Runtime YOLO v11 OpenCV C++ User Study SUS Evaluation Statistical Analysis
